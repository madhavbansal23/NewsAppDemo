# NewsAppDemo
Android News App using MVVM Dagger2 Kotlin Room Livedata Retrofit

- Uses NewApi.org Api to fetch top news<br/>
- Has offline access using Room DB<br/>
- Uses Dagger2 for DI<br/><br/>
